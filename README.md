Interview Questions

1. What is JSX, and why is it used?
   Answer:
   JSX মানে JavaScript XML। এর সাহায্যে আমরা JavaScript ফাইলের ভিতরে HTML এর মতো কোড লিখতে পারি।

2. What is the difference between State and Props?
   Answer:
   State হলো একটি component এর নিজের data। এই data change করা যায় এবং যখন state change হয় তখন component আবার render হয়।
   Props হলো data যেটা এক component থেকে আরেক component এ পাঠানো হয়।
   
3. What is the useState hook, and how does it work?
   Answer:
   useState হলো React এর একটি hook। এটা functional component এ state ব্যবহার করার জন্য লাগে। এটা দুইটা জিনিস দেয়, একটা হলো current value আরেকটা হলো একটা function যেটা দিয়ে value change করা যায়।

4. How can you share state between components in React?
   Answer:
   state parent component এ রাখা এবং props দিয়ে child component এ পাঠানো।
5. How is event handling done in React?
   Answer:
   এখানে event এর সাথে string না দিয়ে function দিতে হয়।
